# C Beginner Projects

This repository contains my beginner C programs for practice.

## Programs
1. even_odd.c - Checks if a number is even or odd
2. add_numbers.c - Adds two numbers
3. area_menu.c - Calculates area of square or circle
4. positive_negative_zero.c - Checks if number is positive, negative, or zero
5. greatest_of_three.c - Finds the greatest of three numbers
